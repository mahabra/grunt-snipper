grunt-snipper

Осуществляет сборку файлов из сниппетов в случаях, когда необходимо создать один большой файл, для удобства разработки разбитый на множество подфайлов. Поддерживает бесконечную вложенность.

Установка

Этот плагин требует node.js и Grunt 
npm install grunt-snipper

Настройка

Пример файла gruntfile.js: 
grunt.initConfig({
	snipper: {
	  development: {
	    files: {
	      '../scripts/': ['./scripts/core.js']
	    }
	  }
	},
	watch: {
		styles: {
			files: ['./scripts/**/*.js'], // which files to watch
			tasks: ['snipper'],
			options: {
				nospawn: true
			}
		}
	}
});
grunt.loadNpmTasks('grunt-snipper');
grunt.loadNpmTasks('grunt-contrib-watch');
grunt.registerTask('default', ['watch','snipper']);
Плагин осуществит сборку файла core.js в директорию /scripts/core.js.
